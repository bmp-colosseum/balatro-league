# Atlas Attribution

Most atlases (Jokers, Tarots, Vouchers, boosters, tags, Enhancers,
chips, 8BitDeck) are extracted from the Balatro game files via
`viewer/scripts/extract_atlases.sh` for use in this viewer.

## Editions.png

Sourced from [EFHIII/balatro-calculator](https://github.com/EFHIII/balatro-calculator)
([MIT License](https://github.com/EFHIII/balatro-calculator/blob/main/LICENSE)).

Provides foil / holographic / polychrome / negative edition overlays in
a single 5×1 atlas (71×94 native tile size).
